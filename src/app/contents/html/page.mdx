# HTML/CSS研修

## 1章 HTML/CSSとは

### 1. HTML

<Kw>HTML</Kw>はHyperText Markup Languageの略であり、<Kw>タグ</Kw>を用いて書式を設定できる<Kw>マークアップ言語</Kw>である。  
<Kw>オープンソース</Kw>で開発されており、**現在のWeb技術の中核を担う**。  
主にWebサイトの大まかな枠組みやコンテンツを作成する目的で使用され、**細かい調整はあまり行わない**。  

### 2. CSS

<Kw>CSS</Kw>はCascading Style Sheetの略であり、**HTMLを装飾**するために使用される。  
タグ単位でスタイルを設定できるのはもちろんのこと、<Kw>クラス</Kw>単位で設定することもできる。  

<Info>

クラスは基本的にCSSで装飾するとき以外は使用しない。  
なお、HTMLにはタグ・クラス以外にIDという要素も設定でき、IDは<Kw>一意の文字列</Kw>でなければならない。  
<Kw>一意の文字列</Kw>とは、割り当てる対象同士で**重複しない文字列**、という意味。  
また、CSSの設定は**タグ\<クラス\<IDの順で上書き適用**されていく。

</Info>
